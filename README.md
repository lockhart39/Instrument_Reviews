# Instrument_Reviews

### About

This is the repository for my NLP project using the "Amazon Musical Instruments Reviews" dataset from Kaggle. Instead of just writing a notebook on Kaggle, I wanted to store the project in a Github repository for reference and to help me create and maintain a Python environment on my local machine for any future projects. 

The link to the dataset is below. 

https://www.kaggle.com/eswarchandt/amazon-music-reviews
